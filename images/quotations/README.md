# Quotation Images Directory

Place your quotation PDF page images in this directory with the following naming convention:

- `page1-cover.jpg` - Cover page with elevator image (will have dynamic text overlay)
- `page2-about.jpg` - About Us page (kept as is)
- `page3-specification.jpg` - Lift specification page background (will have dynamic data overlay)
- `page4-features.jpg` - Features page (kept as is)
- `page5-payment-top.jpg` - Payment details section at top (optional, for payment details header)
- `page7-terms1.jpg` - Terms & Conditions page 1 (optional)
- `page8-terms2.jpg` - Terms & Conditions page 2 (optional)
- `page9-terms3.jpg` - Terms & Conditions page 3 (optional)

## Image Requirements

- Format: JPG/JPEG
- Recommended size: A4 dimensions (595.28 x 841.89 points or 210 x 297 mm)
- Resolution: 300 DPI for best quality

## Notes

- If images are not found, the PDF generator will create a fallback layout with text-only content
- The system will automatically overlay dynamic text on pages 1, 3, 5, and 6
- Pages 2, 4, and terms pages will be embedded as-is if images are provided
